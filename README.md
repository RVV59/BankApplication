# Финансовый аналитический сервис

Этот проект представляет собой финансовый аналитический сервис, который предоставляет пользователю возможность анализировать свои расходы, доходы, отслеживать курсы валют и цены акций, а также использовать дополнительные функции, такие как "Инвесткопилка" и поиск транзакций по телефонным номерам.

## Основные функции

### Веб-страницы
1. **События**:
   - Отображение финансовых событий (расходы, доходы, курсы валют, цены акций) за выбранный период.
   - Генерация JSON-ответа с детализацией по категориям.

2. **Выгодные категории повышенного кешбэка**:
   - Отображение категорий, по которым предоставляется повышенный кешбэк.

### Сервисы
1. **Инвесткопилка**:
   - Позволяет копить деньги через округление трат. Пользователь может выбрать шаг округления (10, 50 или 100).
   - Пример: Покупка на 1712 ₽ округляется до 1750 ₽, и 38 ₽ попадают в "Инвесткопилку".

2. **Простой поиск**:
   - Поиск транзакций по ключевым словам.

3. **Поиск по телефонным номерам**:
   - Возвращает JSON со всеми транзакциями, содержащими в описании мобильные номера.
   - Пример:
     ```json
     [
       {"description": "Я МТС +7 921 11-22-33", "amount": 500},
       {"description": "Тинькофф Мобайл +7 995 555-55-55", "amount": 300}
     ]
     ```

### Отчеты
1. **Траты по категории**:
   - Генерация отчета о расходах по выбранной категории.

### Файл пользовательских настроек
- **`user_settings.json`**:
  - Содержит настройки пользователя, такие как выбранные валюты и акции.
  - Пример:
